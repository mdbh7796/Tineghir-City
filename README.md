# Tineghir City Website

A responsive discovery platform for Tineghir, Morocco, showcasing its natural beauty, cultural heritage, and travel attractions like the Todra Gorge and the palm groves.

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/mdbh7796/Tineghir-City.git
   cd Tineghir-City
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Build the CSS:**
   ```bash
   npm run build
   ```

## Usage

*   **View the Website:** Open `index.html` in any modern web browser.
*   **Admin Panel:** Access `admin.html` to manage site content.
*   **Development:** Use `npm run watch` to automatically compile styles during development.
